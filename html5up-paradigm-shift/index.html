Projects & Experience Highlights

These projects reflect my background in research-driven machine learning and practical data analytics. They span healthcare AI, predictive modelling, dashboarding, and data-driven reporting.

Ophthalmic Disease Detection via Latent Representations

Developed an end-to-end machine learning pipeline for automated ophthalmic disease classification using OCT imaging data. Built U-Net segmentation for RNFL thickness maps, implemented Beta-VAE feature extraction, and trained a multi-label MLP classifier to support disease identification and representation learning.

Predictive Model for Glioblastoma Survival

Developed a machine learning model using neuro-ophthalmic and retinal imaging data to predict patient survival outcomes. Conducted exploratory data analysis, feature selection, leave-one-out cross-validation, and feature importance analysis, contributing to publication-ready methodology and results.
